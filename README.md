# Astro Foundation

## Descripción
Astro Foundation es un proyecto diseñado para aprender las bases del framework.

## Instalación


1. Navegar al directorio del proyecto:
	```sh
	cd astro-foundation
	```
2. Instalar dependencias:
	```sh
	npm install
	```

## Uso

Para iniciar el proyecto, ejecuta:

```sh
npm run dev
```
